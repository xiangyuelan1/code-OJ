📋 系统概览

✨ 升级亮点

基于现有OJ系统，新增知识树管理、智能AI辅助、实时对战系统、完整考试体系、积分等级系统等核心功能，打造专业级编程学习平台。

📚 知识树管理

支持两层知识树结构，AI自动解析文件生成知识体系，智能归类题目到对应知识点。

🤖 AI智能辅助

AI自动生成测试样例、AI生成题解、智能解析上传文件、智能归类题目。

⚔️ 实时对战

1v1排位赛、友谊赛、组队竞技，支持实时得分排行和积分对战系统。

📝 完整考试

作业模式、限时考试、智能监考、考试分析全支持。

🏆 积分系统

难度加权积分、等级称号、成就徽章、对战奖惩机制。

👥 在线状态

实时显示在线人数，在线用户列表，WebSocket即时通信。

📊 系统数据流

1

用户请求

用户通过Web界面发起请求，WebSocket用于实时功能（对战、在线状态），REST API用于常规操作。

2

API网关认证

JWT Token验证、RBAC权限检查、请求限流、CORS跨域处理。

3

业务逻辑处理

根据请求类型分发到对应的服务层（题目、对战、考试、积分、AI等）进行处理。

4

数据持久化

Prisma ORM与数据库交互，Redis缓存热点数据，文件存储处理上传。

5

响应返回

处理结果返回客户端，WebSocket推送实时更新（对战得分、在线人数等）。

⚡ 核心功能模块

📖 题解内嵌流程

1

创建/编辑题目

管理员通过表单创建题目，可选择手动填写题解或使用AI自动生成。

2

AI生成题解 (可选)

点击"AI生成题解"按钮，系统调用AI分析题目并生成完整的解题思路和代码。

3

保存到题目

题解作为JSON字段直接存储在题目记录中，无需单独创建和管理。

🗂️ 知识树管理流程

1

上传文件

支持txt、pdf等格式文件上传，AI自动解析内容提取知识点。

2

AI生成知识树

AI分析文本，生成两层树形结构（一级分类 + 二级知识点）。

3

用户确认编辑

预览生成结果，支持手动调整节点名称、顺序、描述。

4

题目关联

创建题目时可选择关联到指定知识树节点，AI也可自动推荐最合适的节点。

⚔️ 对战系统流程

1

匹配/挑战

选择1v1排位赛或友谊赛，系统基于积分匹配对手，或直接挑战指定用户。

2

实时对战

WebSocket实时同步题目、得分、用时，支持代码编辑器实时答题。

3

结算奖励

根据胜负、正确率、用时计算积分变化，更新排行榜。

📝 考试系统流程

1

创建考试

管理员设置考试标题、时长、题目、是否启用监考等参数。

2

开始考试

学生点击"开始考试"后计时，记录开始时间，启动监考监控。

3

监考监控

检测切屏、复制粘贴、鼠标失焦等行为，记录违规日志，超限自动提交。

4

提交与分析

时间到或手动提交后，计算成绩，生成薄弱点分析，推荐练习题目。

🗄️ 数据模型设计

👤 User (用户)

id UUID
username 用户名
email 邮箱
role ADMIN/STUDENT/TEACHER
points 积分
level 等级
isOnline 在线状态

🌳 KnowledgeTree (知识树)

id UUID
name 节点名称
parentId 父节点ID (可空)
level 层级 (1或2)
order 排序序号

📋 Problem (题目) - 扩展

id UUID
title 题目标题
type PROGRAMMING/CHOICE/FILL_BLANK
solution 内嵌题解 (JSON)
knowledgeTreeId 关联知识树节点
aiGeneratedTestCases AI生成标记

🎯 Match (对战)

id UUID
type 1V1_RANKED/1V1_FRIENDLY/GROUP_ARENA
status WAITING/IN_PROGRESS/COMPLETED
winnerId 获胜者ID
rewards 积分奖励 (JSON)

📄 Exam (考试)

id UUID
title 考试标题
type HOMEWORK/TIMED_EXAM
duration 时长 (分钟)
enableProctoring 启用监考
violations 违规记录 (JSON)

🏆 Achievement (成就)

id UUID
name 成就名称
criteria 达成条件 (JSON)
points 奖励积分
icon 图标URL

🤖 AI功能集成

🧠 AI服务能力

1

知识树自动生成

解析txt/pdf文件，识别知识点，生成两层树形知识结构。

2

题目智能归类

分析题目内容，自动推荐最合适的知识树节点。

3

测试样例生成

根据题目描述，AI生成包含边界条件的测试用例。

4

题解自动生成

5

文件题目导入

解析包含题目的文件，自动提取题目、答案、测试用例。

📅 开发计划

🚀 Phase 1: 基础扩展 (1-2周)

  • 题解内嵌到题目数据模型
  • AI生成测试样例功能
  • AI生成题解功能
  • 文件导入题目（txt/pdf）
  • 基础积分系统

🌳 Phase 2: 知识树系统 (1周)

  • 知识树数据结构和API
  • 知识树管理页面
  • AI解析文件生成知识树
  • 题目关联知识树
  • AI自动归类题目

⚡ Phase 3: 实时系统 (2周)

  • WebSocket集成
  • 在线人数统计和显示
  • 1v1对战系统
  • 排行榜功能
  • 组队竞赛系统

📝 Phase 4: 考试系统 (2周)

  • 考试管理功能
  • 在线考试页面
  • 智能监考功能
  • 考试分析和反馈
  • 考试与积分联动

✨ Phase 5: 完善优化 (1周)

  • 成就系统
  • 等级特权功能
  • 性能优化
  • 安全加固
  • 文档完善

📊 预计工期

总计: 6-8周（可根据团队规模和优先级调整）
